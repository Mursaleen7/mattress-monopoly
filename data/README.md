## üí° Key Principles

### 1. Information First
All psychological triggers are built on accurate, verified information. Never sacrifice accuracy for conversion.

### 2. Progressive Enhancement
Start with v4.0 foundation (geo + weather_profile), then add v5.0 triggers to high-value cities.

### 3. Test and Measure
A/B test everything. What works in Austin might not work in Seattle.

### 4. Mobile First
70%+ of traffic is mobile. One-click actions (Maps, booking) are critical.

### 5. Psychological Leverage
- FOMO (schedule_logic): Loss aversion
- Weather (weather_profile): Risk mitigation
- Price (competitor_comparison): Anchoring effect
